# KABAN:manager

Desktop-приложение для управления проектами, задачами и командой разработчиков (Python + PyQt5, SQLite).

## Требования

- Python 3.10+
- Windows / Linux / macOS

## Установка

```bash
cd KABAN
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

При первом запуске автоматически создаётся база `database/kaban.db` из `database/kaban.sql`.

## Учётные записи по умолчанию

| Логин       | Пароль        | Роль        |
|-------------|---------------|-------------|
| admin       | admin         | Администратор |
| manager     | password123   | Менеджер    |
| developer1  | password123   | Разработчик |

После установки **смените пароль admin** в разделе «Настройки» или «Администрирование».

## Возможности

- Роли: администратор, менеджер, разработчик
- Проекты, задачи, разработчики, отчёты
- Kanban-дашборд, уведомления о дедлайнах
- Экспорт в CSV и Excel
- Светлая / тёмная тема, настройка фона и акцента

## Тесты

```bash
pytest tests/ -q
```

## Структура

- `main.py` — точка входа
- `models/` — модели и БД
- `services/` — бизнес-логика
- `controllers/` — слой контроллеров
- `ui/` — интерфейс PyQt5
- `database/` — SQL-схема и данные
